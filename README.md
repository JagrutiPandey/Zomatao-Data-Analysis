## 📌 Project Overview

This project performs an in-depth analysis of Zomato restaurant data using Python. It uncovers trends in pricing, ratings, online ordering, and restaurant categories through data visualization and exploratory data analysis (EDA).

---

## 🎯 Objectives

* 📊 Analyze restaurant ratings and cost distribution
* 🍔 Identify popular restaurant types
* 📦 Study the impact of online ordering
* 📍 Discover patterns in customer preferences

---

## 🛠️ Tech Stack

* **Python**
* **Pandas** – Data manipulation
* **NumPy** – Numerical operations
* **Matplotlib** – Basic visualization
* **Seaborn** – Advanced visualization

---

## 📁 Dataset Details

The dataset includes:

* Restaurant Name
* Location
* Cost for Two
* Ratings
* Online Order Option
* Restaurant Type

---

## 📊 Analysis & Visualizations

* 📌 Heatmaps for online orders vs restaurant type
* 📌 Count plots and distribution graphs
* 📌 Rating vs cost analysis
* 📌 Trend analysis using charts

---

## 📈 Key Insights

* Restaurants offering online orders tend to attract more customers
* Pricing significantly varies across different restaurant categories
* Higher-rated restaurants often maintain consistent service quality
